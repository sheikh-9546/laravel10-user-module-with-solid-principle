Introduction

Base URL: http://user-module.test 
This documentation aims to provide all the information you need to work with our API.

<aside>As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

OAuth

Allow to log in user

Example request:
curl --request POST \
    "http://user-module.test/api/v1/oauth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"super@module.com\",
    \"password\": \"password\"
}"

const url = new URL(
    "http://user-module.test/api/v1/oauth/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "super@module.com",
    "password": "password"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Permission

APIs for managing permissions

Allow to pull the permissions

requires authentication

Example request:
curl --request GET \
    --get "http://user-module.test/api/v1/permission?search=&page=&per_page=&sort_column=&sort_direction=" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://user-module.test/api/v1/permission"
);

const params = {
    "search": "",
    "page": "",
    "per_page": "",
    "sort_column": "",
    "sort_direction": "",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
